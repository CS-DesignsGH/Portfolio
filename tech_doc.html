JavaScript and Java

Java is an object- oriented, class based programming language designed to have as few dependencies as possible. Java code can be ran on any platform that supports Java.

Javascript is somewhat similar to Java but differs from it at the core. One main difference is that Javascript goes off of a runtime system based on small values, while Java has a compile-time system formed by classes built from declarations.
